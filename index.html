Rust Starters

Contributing to rust is fun!

Sometimes it's hard to know where to get started, though. Rust Starters is a list of easy tasks that are good for beginners.

If you want to work on an issue, make sure to claim it by commenting on it before working on it! Check out our CONTRIBUTING.md for more helpful tips on how to contribute to Rust.

This site's source can be found at https://github.com/g-k/servo-starters
